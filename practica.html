Guía rápida: instalar y usar Git + crear repositorio en GitHub

Archivo con pasos y comandos listos para copiar. Ideal para práctica en clase.

Antes de empezar

Requisitos: Internet · Terminal

Verifica que tienes Git instalado y tu usuario configurado. Aquí están los pasos esenciales (copiar con botón):

Configuración inicial (única vez) 
git config --global user.name "Tu Nombre"
git config --global user.email "tu.email@example.com"
git config --global init.defaultBranch main
Crear repo local desde cero 
mkdir proyecto-demo
cd proyecto-demo
git init
echo "# Proyecto Demo" > README.md
git add .
git commit -m "Primer commit"
Enlazar con GitHub (SSH) 
git remote add origin git@github.com:tu-usuario/proyecto-demo.git
git branch -M main
git push -u origin main
Si usas HTTPS 
git remote add origin https://github.com/tu-usuario/proyecto-demo.git
git branch -M main
git push -u origin main
Crear rama, trabajar y subir 
git checkout -b feature/mi-funcion
# editar archivos...
git add .
git commit -m "feat: agrega mi-funcion"
git push -u origin feature/mi-funcion
Comandos útiles (resumen) 
git status
git log --oneline --graph --all
git pull --rebase origin main
git merge feature/mi-funcion
git branch -d feature/mi-funcion

Práctica sugerida (rápida)

  1. Configura tu nombre y correo (usa la sección Configuración inicial).
  2. Crea la carpeta `proyecto-demo`, inicializa Git y haz un primer commit.
  3. En GitHub (web) crea un repositorio vacío `proyecto-demo` y copia la URL SSH o HTTPS.
  4. Enlaza el remoto y haz `git push -u origin main`.
  5. Crea una rama, modifica un archivo, haz commit y súbela; luego abre un Pull Request en GitHub.

Notas rápidas

  • Si usas Windows, abre Git Bash para evitar problemas de CRLF.
  • Si recibes Permission denied (publickey), revisa tu clave SSH y que esté añadida a GitHub.
  • Para HTTPS usa un Personal Access Token en lugar de contraseña si GitHub lo solicita.

Recursos rápidos

copiar comandos

Comandos para generar y añadir clave SSH (recomendado):

ssh-keygen -t ed25519 -C "tu.email@example.com"
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519
cat ~/.ssh/id_ed25519.pub

Check rápido

  • git --version — verificar Git
  • git config --list — ver config
  • git remote -v — ver remotos
